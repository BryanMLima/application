## Installation

OS used: Manjaro 19.0.2

For this application it was used the Flask framework and pyOpenSLL.
To install it just do the following command:

```bash
pip install flask
pip install pyopenssl
```

To run the application head to "application/api/" folder then run the following command:

```bash
python main.py
```

And then head to this ip:

http://127.0.0.1:5000/
